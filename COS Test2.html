Shipment Overview

Search cargo records, filter by phase, and review each shipment timeline.

Create New Order

Order taking staff can create a new cargo record here before it moves to manufacturing.

Order intake logic: new orders are created in the Inquiry stage first. After that, the assigned team can move them into Manufacturing.

Update Shipment

Use this form to change cargo phase, estimated dates, owner, and remarks.

Workflow control logic: only accounts with permission for the cargo's current workflow stage can amend it. Admin accounts have full access.

Completed Tasks

Archived completed shipments are grouped by year and kept for a minimum of 5 years.

User & Permission Management

Add users, remove users, and decide which workflow stages each account can amend.

Example setup: A can handle Local Logistics (Warehouse → Port), B can handle Shipping, and C can handle Manufacturing. Admin can control all of this here.

Client Account Permission

Create separate client login accounts and bind each account to selected cargo projects.

Example: a client account can be linked only to its own ongoing cargo projects, so the client portal shows only relevant shipment progress.
